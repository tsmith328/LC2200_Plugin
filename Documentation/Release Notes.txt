Version 1.0

Features:

Syntax Highlighting for LC-2200 Source Code
Error checking for LC-2200 Source Code
Automated build support into 16-bit and 32-bit assembly code
Support for LC-2200 Projects containing LC-2200 source files
Builds source files into 16-bit and 32-bit hex files for use in simulator
Quick-fix support for common errors in source code


Known Bugs:

Assembled code view only shows 16-bit code, not 32-bit code
Quick-fix deletes entire instruction when there is an error in one of its fields
If assembled view does not update, try clicking in a different view to kickstart it.
